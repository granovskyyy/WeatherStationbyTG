# WeatherStationbyTG
A simple weather station based on MKL05z32 chip, LCD display, SHT35 temperature, humidity sensor and 4x4 keyboard. 

# MAIN FUNCTIONALITES 
This project provides: 
  - displaying current temperature and humidity
  - changing temperature/humidity limits
  - alerts when temperature/humidity is out of range
